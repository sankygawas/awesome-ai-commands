# Awesome AI Commands

A comprehensive collection of useful AI-powered command-line tools and automation scripts to enhance your development workflow.

## Overview

This repository contains custom statusline scripts, configuration files, and utilities designed to work seamlessly with Claude Code and other AI development tools. The project focuses on providing real-time development environment information and enhancing the CLI experience.

Check the `.claude/` folder for the statusline command and configuration files.

## File Structure

.
├── .claude/
│   └── settings.local.json    # Claude Code configuration
